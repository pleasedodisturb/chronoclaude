privacy policy

Last updated: 2026-04-17

chronoclaude is a local Claude Code plugin. It does not collect, transmit, store, or share any personal information.

  • No telemetry.
  • No analytics.
  • No network requests of any kind.
  • No third-party services.
  • No cookies or tracking (the plugin has no web surface).

The plugin computes timestamps and idle durations on your local machine and writes a small per-session state file (containing only numeric timings) under your Claude Code plugin data directory. Nothing leaves your computer.

The [timing] block the plugin injects into prompts is delivered to Claude by Claude Code itself, under Anthropic's own privacy policy — this plugin adds no separate data flow.

Questions? Open an issue at github.com/pleasedodisturb/chronoclaude.
